[1] Упражнения раздела 3.2.1
1) Упражнение 3.2. Что такое стандартный конструктор?
  Это конструктор по умолчанию, который может быть вызван без агрументов.


2) Упражнение 3.3. Назовите три способа инициализации объекта класса string.
string s; стандартный конструктор.
string s2(s) инициализция строки s2 как копия строки s.
string s3(n, 'a') инициализация строки s3 последовательностью из n символом a 


3) Упражнение 3.4. Каковы значения переменных s и s2?
string s; 
int main() { 
	string s2;
}

s и s2 не имеют данных, пустые строки 


5) Упражнение 3.6. Объясните, как поступают с символами предваряющих пробелов при чтении данных 
в строковую переменную, когда используется функция getline().

getline() не игнарирует пробелы при чтении данных.


8) Упражнение 3.9. Что делает следующая программа? Действительно ли она допустима? Если нет, то почему?
string s;
cout << s[0] << endl;
Программа выводит первый символ, что храниться в строке s.
Это недопустимо, потому в начале создали s с помощью стандартного конструктора, а он создает пустые строки.
