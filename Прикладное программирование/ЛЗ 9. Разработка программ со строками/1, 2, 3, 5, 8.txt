1) Упражнение 3.2. Что такое стандартный конструктор?

Конструктор — это особый тип метода класса, который автоматически вызывается при создании объекта этого 
же класса. Конструкторы обычно используются для инициализации переменных-членов класса значениями, 
которые предоставлены по умолчанию/пользователем, или для выполнения любых шагов настройки, необходимых 
для используемого класса. Конструктор, который не имеет параметров (или имеет параметры, все из которых 
имеют значения по умолчанию), называется конструктором по умолчанию, или стандартным конструктором.

2) Упражнение 3.3. Назовите три способа инициализации объекта класса string.

1. Иницилизация во время объявления, например:
   string str("Hello, world!");

2. Иницилизация путём объявления с присваиванием, например:
   string str = "Hello, world!";

3. Инициализация путём заполнения строки символами c (n - количество символов), например:
   string str(n, c);

3) Упражнение 3.4. Каковы значения переменных s и s2?
string s; 
int main() { 
	string s2;
}

s и s2 - пустые строки

8) Упражнение 3.9. Что делает следующая программа? Действительно ли она допустима? Если нет, то почему?
string s;
cout << s[0] << endl;

Выводит самый первый символ строки s (нумерация символов начинается с 0).
Она недопустима, так как мы не инициализировали строку, значит, при её создании использовался конструктор
по умолчанию, который создаёт строку пустой, а это значит, что в ней не содержится ни единого символа.
