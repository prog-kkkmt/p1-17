№1. Упражнение 3.2. Что такое стандартный конструктор?
Это конструктор по умолчанию, то есть без аргументов.

№2. Упражнение 3.3. Назовите три способа инициализации объекта класса string.
string s1; это стандартный конструктор; s1 - пустая строка
string s2(s1); инициализация строки s2, как копии строки s1
string s3("value"); инициализация строки s3, как копии строкового литерала
string s4(n, 'x'); инициализация строки s4 последовательностью из n символом (элементов) x

№3 Упражнение 3.4. Каковы значения переменных s и s2?
string s; 
int main() 
{ 
	string s2;
}

s и s2 - это пустые строки, так как в них ничего не  вводится.

№4. Упражнение 3.6. Объясните, как поступают с символами предваряющих пробелов при чтении данных 
в строковую переменную, когда используется функция getline().

Функция getline() не игнорирует пробелы при чтении данных в строковую переменную.

№5. Упражнение 3.9. Что делает следующая программа? Действительно ли она допустима? Если нет, то почему?
string s;
cout << s[0] << endl;

Программа выводит 1-ый символ того, что хранится в ячейке памяти выделенной для строки s.
Это недопустимо так как создали объект s с помощью конструктора (по умолчанию), а он создает пустую строку, в которой ничего нет.
