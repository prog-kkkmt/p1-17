[1] Упражнения раздела 3.2.1
1) Упражнение 3.2. Что такое стандартный конструктор?

Ответ: Стандартный конструктор - это конструктор по умолчанию, который может быть вызван без аргументов.

2) Упражнение 3.3. Назовите три способа инициализации объекта класса string.

Ответ: 
string s - стандартный конструктор.
string s1(s) - инициализция строки s1 как копия строки s.
string s2(n, 'k') - инициализация строки s2 последовательностью из n символом k.

3) Упражнение 3.4. Каковы значения переменных s и s2?
string s; 
int main() { 
	string s2;
}

Ответ: s и s2 - пустые строки, не имеют данных.


5) Упражнение 3.6. Объясните, как поступают с символами предваряющих пробелов при чтении данных 
в строковую переменную, когда используется функция getline().

Ответ: Функция getline() не игнорирует предварящие пробелы при чтении данных в строковую переменную. 

8) Упражнение 3.9. Что делает следующая программа? Действительно ли она допустима? Если нет, то почему?
string s;
cout << s[0] << endl;

Ответ: По идеи выводит первый символ в строке s. Это недопустимо, потому что объект s создали с помощью конструктора по умолчанию, а он в свою очередь создает пустые строки.
