Напишите простой класс с именем Person. Этот класс должен иметь:

   Три закрытые переменные "Имя", "Социальная роль", "Пол", "Возраст".

   Открытый метод с именем setValues(), который позволит устанавливать значения для закрытых переменных;

   Открытый метод с именем print(), который будет выводить объект класса Numbers в следующем формате: "Социальная роль" "Имя" "Пол" "Возраст".
