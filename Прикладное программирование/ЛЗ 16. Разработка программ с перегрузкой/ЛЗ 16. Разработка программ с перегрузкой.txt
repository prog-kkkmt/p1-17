Задание 2. [2]Упражнение 7.35. Объясните результат второго объявления в каждом из следующих наборов. 
Укажите, какое из них (если есть) недопустимо.
(a)	int calc(int, int);
	int calc(const int, const int);
(b)	int get(); double get();
(c)	int *reset(int *); double *reset(double *);

	Вариант a. Во втором объявлении передаются константные переменные, вернут функции одно и то же, но передать в первую можно любую целочисленную переменную, а во вторую только константную, неизменяемую

	Вариант b недопустим, т.к. эти две функции не различаются ничем, никакие аргументы не передаются и вызывается ошибка "ambiguating new declaration of 'double get()'"

	Вариант c. Во втором объявлении передается и возвращается указатель на тип double, для работы этой функции не только с целочисленным типом данных
