Пример билета по модулю
Задание для разработки.	Автовокзал. 
Автобусы производят рейсы до станций по расписанию. Каждый автобус вмещает не более определенного количества пассажиров
Таблицы: станции (код станции, название станции), автобусы (код автобуса, марка автобуса, государственный номер, вместимость), рейсы (код рейса, код станции, код автобуса, время отправления).
Определить:
- сколько выполняется рейсов до каждой станции?
- каково общее количество пассажиров?

1. По текстовому описанию задачи построить диаграмму прецедентов
ПК 1.6	Разрабатывать компоненты проектной и технической документации с использованием 
графических языков спецификаций. 0.20 - 0.22 (2 минуты)

2. По описанию и диаграмме прецедентов построить диаграмму классов
ПК 1.1	Выполнять разработку спецификаций отдельных компонент. 0.22 - 0.26 (4 минуты)

3. Разработать определения классов по диаграмме
ПК 1.2	Осуществлять разработку кода программного продукта на основе готовых спецификаций на уровне модуля. 0.05 - 0.20 (15 минут)

4. Разработать проект с классами для выполнения задания
ПК 1.2	Осуществлять разработку кода программного продукта на основе готовых спецификаций на уровне модуля. 23:05 - 0.05 (1 час)

5. Показать отладку одного из модулей при разработке
ПК 1.3	Выполнять отладку программных модулей с использованием 
специализированных программных средств. 0.27 - 0.30 (3 минуты)

6. Подготовить тестовые наборы и провести тестирование одного из модулей
ПК 1.4	Выполнять тестирование программных модулей. 0.30 - 0.44 (14 минут)

Указать значение O для оптимального кода
ПК 1.5	Осуществлять оптимизацию программного кода модуля. 0.44 - 0.47 (3 минуты)

Алгоритм поиска по базе данных = O(n), т.к. алгоритм должен пройти все строки, чтобы найти нужные

Итог: 1 час 41 минута (101 минута)

Нормативное время выполнения - 120 минут
Критерии оценки 0..5 баллов
Для получения оценки "3" требуется набрать не менее 3 баллов по каждой ПК
Для получения оценки "4" требуется набрать не менее 24 баллов
Для получения оценки "5" требуется набрать не менее 31 балла
