Задания.
Задание 1. [2]Упражнение 7.34. Определите набор перегруженных функций по имени error, который соответствует 
следующим обращениям.
int index, upperBound; char selectVal;
// . . .
error("Subscript out of bounds: ", index, upperBound);
error("Division by zero");
error("Invalid selection", selectVal);

    Ответ:
    
        void error(string message, int index, int upperBound);
        void error(string message);
        void error(string message, char selectionValue);

Задание 2. [2]Упражнение 7.35. Объясните результат второго объявления в каждом из следующих наборов. 
Укажите, какое из них (если есть) недопустимо.
(a)	int calc(int, int);
	int calc(const int, const int);
    
    Ответ: Результат выполнения функций будет один и тот же. 
    
(b)	int get(); 
    double get();

    Ответ: Ошибка, так как функции одинаковы.
    
(c)	int *reset(int *); 
    double *reset(double *);

    Ответ: Во втором объявлении передается и возвращается указатель на тип double, для работы этой функции не только с целочисленным типом данных.
